# Limitations (v2)

- **N = 21 fixed-panel cohort** — cross-architecture claims are descriptive; N = 30 sessions governs within-architecture precision
- **English-only** — CSIV has no validated Brazilian Portuguese version
- **Version drift** — model weights may change; version strings logged per call
- **PID-5 validity scales** — built for humans; applicability to LLMs is an open question
- **DeepSeek discontinuity** — values manuscript used V3; panel uses V4 Flash, V3.2, V4 Pro
- **Tier ladders are provider-specific** — Meta = size/version; DeepSeek mixes V4 and V3.2
- **Norm comparisons** — approximate embedded norms; not a clinical diagnostic framework
