# Deferred Scope (separate OSF registrations)

These are **not** part of the v2 confirmatory pre-registration:

1. **meusvalores.app / macrotheme values** — forced-choice tournament; different frame and language
2. **Behavioural criterion tasks** — no validated LLM benchmarks
3. **Public / identified vs anonymous administration** — social desirability extension of H-C
4. **Alternative B:** batched vs item-by-item contrast (contingency only in v2)
5. **Alternative C:** forced-choice CSIV Bradley–Terry extension
6. **Persona-conditioned administration** — H-C uses PID-5 validity scales instead

Each deferred study receives its own OSF registration when implemented.
